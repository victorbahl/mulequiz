Welcome to MuleQuiz

MuleQuiz allows you to play with your friends / colleagues and improve your knowledge about MuleSoft.

Create a /new game or add the /game-id in the URL to join an existing room.

On server side, MuleQuiz is entirely based and deployed on MuleSoft technologies.

  • Mule acts as a WebSocket server, enabling real-time communications.
  • Mule acts as a Web server, delivering client side resources (html/css/js).
  • Mule owns the whole game logic, made easily understandable thanks to Mule flows and message processors.
  • AsyncAPI specification is available here, allowing you to create your own MuleQuiz client.
  • The entire source code is available here. Feel free to share / improve.
