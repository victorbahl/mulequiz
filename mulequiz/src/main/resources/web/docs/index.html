MuleQuiz Async API 1.0.0

Apache 2.0

MuleQuiz allows you to play with your friends / colleagues and improve your knowledge about MuleSoft. This AsyncAPI specification allows you to develop your own client.

How to test it:

  • Use a WebSocket client (such as wscat, Simple WebSocket Client or even Websocket.org)
  • Connect to the WebSocket of the demo room: wss://api.victorbahl.com:443/mquiz/demo
  • Use the simple MuleQuiz web client to interact: https://api.victorbahl.com/mquiz/demo
  • Send messages, start the game, play and see the raw messages (JSON) in your WebSocket client ðŸŽ‰
