Welcome to MuleQuiz

MuleQuiz allows you to play with your friends / colleagues and improve your knowledge about MuleSoft. MuleQuiz is entirely based on MuleSoft technologies + a little a bit of HTML/JS/CSS on client side.

  • Path to join an existing game room: /my-room-name (eg. /ipl-uun-gad)
  • Path to create a new game room: /new
